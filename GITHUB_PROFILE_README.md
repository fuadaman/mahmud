## 🚀 Featured Projects

### 🛒 [Where's My Tofu?](https://www.wheresmytofu.me/)
> Grocery price comparison app with an AI-powered store flyer scanner

A full-stack web app that helps budget-conscious shoppers find the cheapest prices across nearby stores.
Users create shopping lists, compare prices across multiple stores, and use an AI-powered flyer scanner to automatically extract product prices from store flyers (チラシ).

**Highlights:**
- 🤖 AI flyer scanner powered by **Groq / Llama 4 Scout** with Google Gemini fallback
- 🗺️ Store maps & optimised shopping routes via **Mapbox GL JS**
- ⚙️ Background flyer processing with **Solid Queue**
- 👥 Team project (4 members)

**Tech:** `Rails 7.1` `Hotwire` `PostgreSQL` `RubyLLM` `Mapbox GL JS` `Cloudinary` `Solid Queue` `Devise` `Docker`

### 💊 [mAi Kusuri (まい薬)](https://mai-kusuri-6f1cdb2a3fdb.herokuapp.com)
> AI-powered OTC medicine assistant for tourists in Japan

Helps tourists find OTC medications and nearby clinics by describing symptoms in plain English.
Features **mAi-Kun**, an LLM-powered chatbot that acts as a virtual pharmacist in a Japanese drugstore.

**Highlights:**
- 🤖 LLM chatbot (**Groq**) for symptom-based medicine recommendations
- 💊 ~1,000 Japanese OTC drugs sourced via **Rakuten Ichiba API**
- 📍 Clinic & pharmacy geolocation via **Nominatim / OpenStreetMap**
- 👥 Team project (4 members)

**Tech:** `Rails 7.1` `Hotwire` `PostgreSQL` `ruby_llm` `Devise` `Nominatim` `Rakuten API` `Bootstrap 5` `Docker`
